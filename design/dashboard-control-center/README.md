# Aura Console design reference

Extracted from the user-provided `Dashboard Design.zip`.

- `Aura Console.dc.html` is the source layout and visual reference.
- `support.js`, `.thumbnail`, and `uploads/` are supporting source assets.
- The production Control Center reimplements the design through React and CSS
  in `packages/ui`; these source files are not bundled into the desktop app.

The adapted interface keeps the original dark operations-console direction,
gold energy accents, compact metrics, and glowing panel treatment while
replacing the sample dashboard content with SuperSaiyan repository, board,
feature, run, and terminal state.
